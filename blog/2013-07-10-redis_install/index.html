Redisインストール

2013-07-10
環境
[CentOS] 5.6 (32bit版カーネル)
インストール
# ソース取得。
# wget http://redis.googlecode.com/files/redis-2.6.7.tar.gz
#
# tar xzf redis-2.6.7.tar.gz
#
# cd redis-2.6.7
#
# 続いてmakeを実行すると下記エラーが発生。
# make
