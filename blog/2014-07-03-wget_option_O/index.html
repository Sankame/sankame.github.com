wget ファイル名を指定してダウンロード

July 3, 2014

1 minute read

-O オプションを使って、ダウンロードファイル名を指定します。

  • 下記例では MyWget という名前に変更。

    # wget -O MyWget http://ftp.gnu.org/gnu/wget/wget-1.15.tar.gz
# ls
# -rw-r--r--  1 root  root  3417936  1月 19 19:17 MyWget
