MySQLコマンド メモ

August 26, 2013

1 minute read

MySQLコマンド メモ WebサーバとDBサーバが1台のマシンにある場合によく行う操作。 # そのマシンのMySQLサーバにrootで接続 # (パスワードを聞かれるので入力してEnter) # mysql -h 127.0.0.1 -u root -p mysql> – UTF8でDBを作成し、接続ユーザを作成 (先頭がtestの文字は読み替えて下さい) mysql> create database testdb default character set utf8; mysql> grant all on testdb.* to ‘testuser’@‘localhost’ identified by ‘testpassword’; mysql> flush privileges; mysql> – 別マシンからDBへ接続する必要がある場合は、以下のように別ユーザを作成 mysql> – (192.168.0.1の部分を別マシンのIPに読み替える) mysql> grant all on testdb.* to testuser@192.168.0.1 identified by ‘testpassword’; mysql> – 作成したDBに切り替え、テーブルを作成 mysql> use testdb; mysql> DROP TABLE IF EXISTS users; mysql> CREATE TABLE users ( mysql> id bigint unsigned NOT NULL AUTO_INCREMENT PRIMARY KEY, mysql> username varchar(32) NOT NULL mysql> ) ENGINE=InnoDB;

Nexus7とGitHubでプログラミング環境を作る

Nexus7とGitHubでプログラミング環境を作る

August 11, 2013

1 minute read

0.環境 [端末] Nexus7 [OS] Android 4.2.2 [BlueToothキーボード] ELECOM TK-FBP013 1.Nexus7にBit Web Serverをインストール (ダウンロード) Android上でPHPとMySQLを動かせるアプリ。(有料 約230円) root化しないで使用可能。 2.Nexus7にTerminal IDEをインストール (ダウンロード) Android用の開発環境。java、c、c++等のコンパイラを含む。 root化しないで使用可能。 3.Hello World Nexus7にBlueToothキーボードを接続。 Nexus7でTerminal IDEを起動。 いくつかのボタンが表示されるので、[Install System]をクリック。 Overwrite ALLにチェックを入れ、[Install System]をクリックし、システムアップデート。 [Terminal IDE]をクリック。 コンソールが起動するので、ファイルを作成し、以下の内容を書き込む。 $ cd $HOME/sdcard/wwww $ $ vim hello.php {% gist 29b89884b38df722f9c6 test_hello.php %} Nexus7でBit Web Serverを起動。 [Localhost]をクリック。 ブラウザが起動し、表示されたファイル一覧から上記で作成したhello.phpをクリック。 ブラウザにhelloと表示されれば成功。 4.GitHubにファイルをアップ(準備) せっかくなのでNexus7からGitHubにファイルをアップしてみる。 $ 公開鍵と秘密鍵を作成。 $ Nexus7で以下のコマンドを打つ。 $ dropbearkey -t rsa -f $HOME/.ssh/id_rsa $ $ 作成した鍵を表示。 $ dropbearkey -y -f ~/.
