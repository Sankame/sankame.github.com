GithubからBitbucketへソース移行

GithubからBitbucketへソース移行

June 21, 2014

1 minute read

0.環境 [CentOS] 5.6 (32bit版カーネル) [Git] 1.9.4 1.ダウンロード Githubのソース取得

任意のディレクトリへ移動。 GitHubのリポジトリから対象ソースを取得。(上図の画面右下がURL)

git clone git@github.com:Sankame/repo-of-github.git

取得したソースの第1階層へ移動。

cd repo-of-github/

Bitbucketへソースアップ 予めBitbucket上で任意のリポジトリを作っておきます。

空のリポジトリができました。

上記で GitHub から取得したソースは origin がGitHubになっているので、 Bitbucket へ変更。

git remote -v origin git@github.com:Sankame/repo-of-github.git (fetch) origin git@github.com:Sankame/repo-of-github.git (push) # git remote rm origin # git remote add origin git@bitbucket.org:Sankame/repo-of-bitbucket.git # git push -u origin –all # pushes up the repo and its refs for the first time # git push -u origin –tags # pushes up any tags GithubのソースがBitbucketへ取り込まれました。
