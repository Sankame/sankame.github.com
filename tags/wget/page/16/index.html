0.環境 [CentOS] 5.6 (32bit版カーネル) [Apache] 2.2.3 [PHP] 5.4.17 [Revive AdServer] 3.0.4 1.インストール 1-0.ダウンロード 適当なディレクトリにソースをダウンロードし、Apache公開ディレクトリへ移動。 (公開ディレクトリは自分の環境に読み替えて下さい)

wget http://download.revive-adserver.com/revive-adserver-3.0.4.tar.gz # tar xzf revive-adserver-3.0.4.tar.gz # mv revive-adserver-3.0.4 revive-adserver # mv revive-adserver /app/. 1-1.インストール開始

ブラウザから、上記インストール先URLへアクセス。 (例) http://localhost/revive-adserver ウィザードに従ってインストールを進めます。

“I Agree >>“ボタンを押すと、”File Permissions - 7 errors” というエラーが表示されました。 画面に表示されたコマンドをそのままコンソールで実行。

chmod -R a+w /app/revive-adserver/var # chmod -R a+w /app/revive-adserver/var/cache # chmod -R a+w /app/revive-adserver/var/plugins # chmod -R a+w /app/revive-adserver/var/templates_compiled # chmod -R a+w /app/revive-adserver/plugins # chmod -R a+w /app/revive-adserver/www/admin/plugins # chmod -R a+w /app/revive-adserver/www/images 再びインストール画面へ戻って”Retry”を押すと、次のステップへ進めます。
